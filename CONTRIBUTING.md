# Contributing to File Manager Pro

Thank you for your interest in contributing to File Manager Pro!

## Development Environment

### Prerequisites
- **Android Studio** Flamingo (2022.2.1) or newer
- **Android SDK** 34 (Android 14)
- **Java** 17 or newer
- **Git** for version control

### Setup Development Environment
1. **Fork the repository**
2. **Clone your fork**
   ```bash
   git clone https://github.com//file-manager-pro.git
   cd file-manager-pro
